# 📊 Social Media Algorithms and Political Polarization

## 🧠 Overview

This research explores how social media algorithms influence political polarization by promoting selective content to users. The study analyzes how recommendation systems and engagement-based ranking contribute to echo chambers and the spread of misinformation. Additionally, a machine learning model is developed to identify and classify polarized or misleading content.

---

## 🎯 Objectives

* Analyze the role of social media algorithms in shaping user opinions
* Detect patterns of misinformation and fake news
* Measure the impact of algorithmic filtering on political polarization
* Build a machine learning model for content classification

---

## 📂 Dataset

* Source: Public datasets (Twitter / social media data)
* Data Type: Text posts, user interactions, engagement metrics
* Preprocessing:

  * Text cleaning (removal of stopwords, punctuation)
  * Tokenization
  * Lemmatization
  * Feature extraction (TF-IDF / embeddings)

---

## ⚙️ Machine Learning Model

### 🔹 Model Type

* Supervised Learning (Classification)

### 🔹 Algorithms Used

* Logistic Regression
* Random Forest
* Support Vector Machine (SVM)
* (Optional) Deep Learning (LSTM / BERT)

### 🔹 Features

* Text-based features (TF-IDF)
* Sentiment scores
* Engagement metrics (likes, shares)

### 🔹 Output

* Classification labels:

  * Neutral
  * Polarized
  * Misinformation

---

## 📈 Results

The model demonstrates effective detection of polarized and misleading content, showing how algorithm-driven exposure influences user perception.

4. Train the model


